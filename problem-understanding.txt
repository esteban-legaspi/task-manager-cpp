Understand the problem: 

project: Task manager

requirements: 
- Structs or class
- vector
- enum
- small database
- well-separated functions
- user input validation
- interactive menu
- clear logic

Data Model: 
- id (int)
- title (string)
- description (string)
- priority (enum: LOW, MEDIUM, HIGH)
- status (boolean: 0 : not done; 1: done)

Minimum features: 
- Create a task
- List all tasks
- View aa task by ID
- Edit a task
- Delete a task

Optional Extensions: 
- Filter tasks by priorty or status.
- Sort tasks (by priorty or ID).
